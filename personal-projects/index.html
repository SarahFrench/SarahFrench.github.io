BeerBot

A Slack bot to organise who buys beers for Friday afternoons at the office

  • Posts to a Friday Beers channel in Slack using the Slack API
  • Uses the Google Sheets API to read/write info about whose turn it is to buy beer
  • Could use membership info for the beers channel to inform who should be included in the beer rota document in Google Sheets
  • Can remind the person due to buy beer on a given Friday, and post "Beer time!" at 4pm in the channel

Mumsnet Birthdays & Anniversary announcements

  • Uses the Hibob API to access HR information, including birthdays and work anniversaries
  • Uses the Google Slides API to read/write info to slides about whose birthday/work anniversary is today
  • The slides will update every working day, and be displayed on a TV screen at te front of the office

PageSpeedBot

A Slack bot to monitor and post updates about the page load speed of Mumsnet websites

  • Either accesses data collected via Google Analytics (?) or use Google's PageSpeed Insights API to get info about pagespeed on the Mumsnet website
  • Posts to a Page Speed channel in Slack using the Slack API
  • The bot could eventually be made to respond to input (e.g. query about a subsection of the website) and give info on an as-needed basis
